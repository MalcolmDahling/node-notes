<h2>Notes</h2>
Frontend created with Create-React-App and Typescript
<br>
Backend created with Express-Generator
<br>

<h2>How to start frontend</h2>
- npm i
<br>
- npm start

<h2>How to start backend</h2>
- npm i
<br>
- nodemon start

<h2>Packages used frontend:</h2>
- Axios
<br>
- Nanoid
<br>
- React-Router-Dom
<br>
- Sass
<br>
- TinyMCE
<br>

<h2>Packages used backend:</h2>
- Cors
<br>
- Mysql2
<br>
